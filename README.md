Lomanai
=======

## Usage

```sh
lomanai --species 'Mus musculus' --species 'Homo sapiens'
#> Mammalia
#> ++Rodentia
#> | \-Mus musculus
#> \+Primates
#>   \-Homo sapiens
-----------------------------------
```
